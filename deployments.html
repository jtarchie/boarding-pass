4 Deployments

The Concourse team maintains a few deployments. The day will eventually come when you have to do something to one of them. This page is here for you to remember when you're panicking.

4.1 Docs (concourse-ci.org)

The Concourse Docs is the face of Concourse, providing the landing page and all documentation for the latest version of Concourse.

The docs are automatically built from the concourse/docs repo and deployed via GitHub pages whenever a new version of Concourse is released.

4.2 CI (ci.concourse-ci.org)

CI is our Concourse instance that builds, tests, and ships Concourse itself. It is publicly viewable and also hosts a handful of example pipelines which are referenced from the documentation here.

4.4 Wings (wings.pivotal.io)

Wings is the precursor to Hush-House. It is deprecated and will be removed soon (probably by March 2020). You probably don't have to know anything about it, but as long as the environment still exists this page will at least acknowledge it.

4.5 Duty Free (resource-types.concourse-ci.org)

Duty Free is a deployment of concourse/resource-types-website, developed and shipped by the Duty Free team as a central repository of community-submitted (and supported) resource types.

I dunno how to deploy it, ask Bishoy! (...and then teach me so I can update this!)

4.6 Cadet (project.concourse-ci.org)

Cadet is a deployment of vito/cadet, a side-project for making our day-to-day interactions with GitHub a bit easier.

It is currently deployed and maintained by Alex willy-nilly. (I'll automate it eventually! ...I promise!)
